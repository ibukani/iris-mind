# Iris Project

Iris is an autonomous AI assistant that learns and evolves. Built with Python, running on Ollama (Qwen3.5 9B).

## Agent Context

Detailed instructions are in `.agent/`:
- `.agent/rules.md` — Project rules, conventions, dependencies
- `.agent/project.md` — Architecture overview, module APIs, data flow
- `.agent/tasks.md` — Task tracking
- `.agent/context.md` — Session context and decisions

## Quick Start

```bash
pip install .  # or: pip install ollama pydantic pyyaml rich chromadb
ollama pull qwen3.5:9b
python main.py
```
