# Iris プロジェクト コンセプト

## プロジェクトの目的

自律的に行動し、進化できるAI"IRIS"を開発する。IRISはデジタルメイドである。

## 目標

1. **ユーザー意図の推測と操作提案**
   - ユーザーの行動パターンを学習し、先回りした提案を行う
   - 提案頻度・方法はユーザーが制御可能

2. **ユーザー指示に基づく操作実行**
   - あらゆる操作に対応できる柔軟なツールシステム
   - Capability Registry による動的な機能拡張

3. **自然な会話**
   - autoモード（デフォルト）: 複雑性に応じて小/大モデルを自動切替、軽快なキャラ会話
   - deepモード: 大モデル強制 + CoT + 深い推論・コード生成
   - stepwiseモード: 大モデル強制 + CoT + タスク分解と計画実行

4. **個としての人格**
   - Neuro-sama を参考にした、知的だけど少しズレた感じのキャラクター
   - ユーモアと親しみやすさ
   - セッションを超えて一貫した人格

## 設計原則

- **ローカルAI優先**: Ollama + Qwen3.5 をベースに、モデル切り替えを容易に
- **自己拡張可能**: Capability Registry によりAI自身が操作範囲を拡大できる
- **ユーザー主権**: すべての変更はユーザーの承認を得る
- **段階的実装**: AGIは目指さず、現実的に可能な範囲から積み上げる
